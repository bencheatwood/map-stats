# Map Stats

Compare two teams within a matchup for the CS2 Major tournament. Uses current season rankings and map stats from HLTV to aid in matchup simulation.

## Features

### Match Simulation

Select the winning team for each matchup, and simulate results for future rounds. The teams chosen dynamically update the comparison stats for future matchups. Seeding for future rounds will be automatically determined based upon VRS ranking and the Buchholz system.

### Stat Comparison

This tool uses each team's map stats from the current season to compare win rate, pick rate, and ban rate for each map in the current map pool, as well as head-to-head stats from their match history.
